# STM32CubeMX Projest

Here you should paste your project, generated by STM32CubeMX.

It Project Manager:
**Toolchain / IDE** set to **Makefile**

Structure of directory should look like:
 - Drivers
 - Inc
 - Src
 - CMakeLists.txt
 - Makefile
 - settings.cmake
 - startup_stm32fXXXXX.s
 - STM32FXXXXXXXX_FLASH.ld
 - toolchain.cmake
 - project_name.ioc
 - .mxproject
